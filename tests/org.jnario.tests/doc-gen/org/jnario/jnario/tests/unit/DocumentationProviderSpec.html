DocumentationProvider

  • returns null if no comment

    documentation(null) => nullValue

  • returns comment

    documentation("Hello World") => "Hello World"

  • '/*' can be escaped with ''

    documentation("Hello \\/*World") => "Hello /*World"

  • '\/*' can be unescaped with ''

    documentation("Hello \\\\/*World") => "Hello \\/*World"

  • '*/' can also be escaped with ''

    documentation("Hello \\*/World") => "Hello */World"

  • '\*/' can also be unescaped with ''

    documentation("Hello \\\\*/World") => "Hello \\*/World"
