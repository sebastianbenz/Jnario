XtendMethodBuilder

  • creates method with specified name

    generatedMethod.is('''
  def myMethod(){
  }
''')

  • adds method parameters

    setContext('''
  describe "Something"{
    fact myMethod(true, 4)
  }
''')
generatedMethod.is('''
  def myMethod(boolean b, int i){
  }
''')

  • returns false for boolean

    setContext('''
  describe "Something"{
    boolean b = myMethod()
  }
''')
generatedMethod.is('''
  def myMethod(){
    return false
  }
''')

  • returns default value for numbers

    setContext('''
  describe "Something"{
    int i = myMethod()
  }
''')
generatedMethod.is('''
  def myMethod(){
    return 0
  }
''')

  • returns null for objects

    setContext('''
  describe "Something"{
    Object o = myMethod()
  }
''')
generatedMethod.is('''
  def myMethod(){
    return null
  }
''')
